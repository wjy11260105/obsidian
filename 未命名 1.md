## 冒泡排序 (Bubble Sort)

```python
def bubble_sort(arr):
    """
    冒泡排序算法
    时间复杂度: O(n²)
    空间复杂度: O(1)
    """
    n = len(arr)
    
    # 遍历所有数组元素
    for i in range(n):
        # 标志位：如果没有交换，说明已排序
        swapped = False
        
        # 最后 i 个元素已排序，不需要比较
        for j in range(0, n - i - 1):
            # 比较相邻元素
            if arr[j] > arr[j + 1]:
                # 交换
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        
        # 优化：如果没有交换，数组已排序
        if not swapped:
            break
    
    return arr


# 测试代码
if __name__ == "__main__":
    # 测试用例
    test_cases = [
        [64, 34, 25, 12, 22, 11, 90],
        [5, 1, 4, 2, 8],
        [1],
        [1, 2, 3, 4, 5],
        [5, 4, 3, 2, 1]
    ]
    
    for test in test_cases:
        original = test.copy()
        result = bubble_sort(test)
        print(f"原数组: {original}")
        print(f"排序后: {result}\n")
```

### 算法说明
- **原理**：通过重复遍历列表，比较相邻元素，将较大的元素"冒泡"到末尾
- **优化**：使用 `swapped` 标志检测是否已排序，避免不必要的遍历
- **适用场景**：小型数组、教学用途
