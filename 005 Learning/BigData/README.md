# 大数据核心技术

本文件夹包含大数据工程师必学的核心技术笔记，包括：

- **Spark** — 分布式计算框架
- **Hadoop** — 分布式存储与计算生态
- **Hive** — SQL on Hadoop
- **Flink** — 流处理框架
- **Kafka** — 消息队列
- **Presto/Trino** — 分布式 SQL 查询引擎
- 其他大数据技术栈

## 快速开始

使用 `Tech Note Template` 创建新的技术笔记。

## 笔记组织

- 按技术名称组织笔记（如 `Spark 核心概念.md`）
- 使用 `topic` 字段标记所属技术
- 通过 `progress` 字段追踪掌握度
- 使用 `related` 字段建立笔记间的关联
