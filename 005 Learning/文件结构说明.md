## 📊 新建文件详细说明

### 模板文件（4 个）

#### 1. Tech Note Template.md
**位置：** `001 Templates/Tech Note Template.md`  
**用途：** 技术概念笔记模板  
**自动保存位置：** `005 Learning/BigData/`  
**关键字段：**
- `topic` — 技术主题（Spark、Flink 等）
- `difficulty` — 难度（beginner/intermediate/advanced）
- `status` — 学习状态（learning/reviewing/mastered）
- `progress` — 掌握度（0-100%）
- `source` — 学习来源
- `related` — 关联笔记

#### 2. Snippet Template.md
**位置：** `001 Templates/Snippet Template.md`  
**用途：** 代码片段模板  
**自动保存位置：** `005 Learning/Snippets/`  
**关键字段：**
- `language` — 编程语言（Python/SQL/Shell/Scala/Java）
- `topic` — 所属主题
- `description` — 片段用途

#### 3. Learning Log Template.md
**位置：** `001 Templates/Learning Log Template.md`  
**用途：** 学习日志模板  
**自动保存位置：** `002 Journal/`  
**关键字段：**
- `topic` — 今日学习主题
- `duration` — 学习时长（分钟）
- `difficulty` — 难度（easy/medium/hard）
- `mood` — 学习状态（1-5）
- `summary` — 一句话总结

#### 4. Tech Case Template.md
**位置：** `001 Templates/Tech Case Template.md`  
**用途：** 技术案例模板  
**自动保存位置：** `005 Learning/Cases/`  
**关键字段：**
- `topic` — 技术主题
- `tech-stack` — 技术栈列表
- `difficulty` — 难度（beginner/intermediate/advanced）
- `progress` — 完成度（0-100%）
- `start-date` — 开始日期
- `deadline` — 截止日期

## 📈 数据流向

```
创建新笔记
    ↓
选择模板（Tech Note/Snippet/Learning Log/Tech Case）
    ↓
填写 Frontmatter（topic, difficulty, progress 等）
    ↓
Templater 自动移动到对应文件夹
    ↓
Learning Dashboard 自动读取并统计
    ↓
显示在对应的表格/图表中
    ↓
用户可视化查看学习进度
```

## 🎯 使用流程

### 学习新技术的完整流程

```
Day 1: 学习新概念
  ├─ 创建技术笔记（Tech Note）
  ├─ 填写 topic: "Spark"
  ├─ 设置 progress: 20, status: learning
  └─ 自动保存到 005 Learning/BigData/

Day 2-5: 深入学习
  ├─ 每天记录学习日志（Learning Log）
  ├─ 保存有用的代码片段（Snippet）
  ├─ 更新技术笔记的 progress 字段
  └─ Learning Dashboard 自动更新统计

Day 6: 复习总结
  ├─ 回顾学习日志
  ├─ 更新技术笔记为 status: reviewing
  ├─ 补充遗漏的知识点
  └─ 整理最佳实践

Day 7: 掌握验证
  ├─ 完成一个技术案例（Tech Case）
  ├─ 更新技术笔记为 status: mastered
  ├─ 设置 progress: 100
  └─ 记录案例笔记和经验
```
