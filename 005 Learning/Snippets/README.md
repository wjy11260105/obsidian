# 代码片段库

本文件夹包含可复用的代码片段，按编程语言分类：

- **Python** — 数据处理、机器学习等
- **SQL** — Hive SQL、Spark SQL、Presto SQL 等
- **Shell** — Linux 命令、脚本等
- **Scala** — Spark Scala API
- **Java** — 大数据框架 Java API

## 快速开始

使用 `Snippet Template` 创建新的代码片段。

## 片段组织

- 使用 `language` 字段标记编程语言
- 使用 `topic` 字段标记所属技术主题
- 在 `description` 字段简述片段用途
- 包含参数说明和使用示例
- 记录常见的坑和注意事项

## 使用方式

在 Learning Dashboard 中按语言浏览所有片段，快速查找和复用。
